Project
========

project solves the problem of where to start with documentation
by providing a basic explanation of how to do it easily:

```python
import project
# Get your stuff done
project.do_stuff()
```


Features
--------

- Be awesome
- Make things faster


Installation

Install $project by running:
```

```
